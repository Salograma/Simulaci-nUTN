# Proyecto de Simulación de numeros pseudoaleatorios

Este proyecto tiene como objetivo implementar y analizar diferentes métodos de generación de números pseudoaleatorios. Se han implementado los siguientes métodos:

1. Método Congruencial Lineal (LCG)
2. Método de Itamaracá (ITA)
3. Método random de Python

## Requisitos

- Python 3.x
- Librerías: numpy, matplotlib, scipy

```bash
pip install numpy scipy matplotlib
```

## Uso

Para ejecutar el proyecto, simplemente ejecute el archivo `main.py`:

```bash
py main.py
```
